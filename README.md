# **c**on**s**ole.**b**ad**g**e()

Print badges in your console.

[![npm version](https://badgen.net/npm/v/csbg)](https://npm.im/csbg) [![npm downloads](https://badgen.net/npm/dm/csbg)](https://npm.im/csbg)

## Install

    npm i csbg

## Usage

```js
import "csbg/install"

console.badge("npm", "v1.0.0")
```
